Nav

class nav-logo เอาไว้ใส่รูปโลโก้ ต้องอยู่ใน class nav-brand
class nav-burger เอาไว้แสดงตัวปุ่มเมนูขนาดเล็ก จะขึ้นมาเมื่อขนาดหน้าจอเล้กลง ปรับแต่งได้ในไฟล์ _initial.scss
class nav-start เป็น div ชั้นนอกเอาไว้จัดชุดเมนูให้อยู่ทางซ้าย
class nav-end เป็น div ชั้นนอกเอาไว้จัดชุดเมนูให้อยู่ทางขวา
class nav-item ต้องใส่เอาไว้ทุกอันที่เป็นตัวเลือกใน menu
class level1 คือชั้นนอกสุด level2 คือชั้นในภายใต้ class nav-dropdown
class nav-divider ใน hr เอาไว้แบบเมนูใน dropdown
