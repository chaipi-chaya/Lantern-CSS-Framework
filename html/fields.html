Label

    <input class="input" type="text" placeholder="Text input">>

    <label class="label">Name</label>

Input

มีแบบธรรมดา

    <input class="input" type="text" placeholder="Text input">>
</div>
และแบบสีต่างๆโดยเพิ่ม class primary, secondary, link, info, success, warning และ danger หลัง class input 

<div class="field">
    <input class="input success" type="text" placeholder="Text input">
    <input class="input" type="text" placeholder="Text input"disabled >
</div>
